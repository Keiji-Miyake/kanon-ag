# Specification - Project Scaffold Skill

## 背景

新規プロジェクトを開始する際、AI駆動開発に適した環境（AGENTS.md、ディレクトリ構造、基本スキルの選定）を手動で構築するのは手間がかかり、設定の漏れが生じる可能性がある。

## 目標

AIが自律的に、またはユーザーの指示により、標準化されたAI駆動開発環境を迅速に構築できるようにする。

## 要件

- **AGENTS.mdの生成**: プロジェクト全体の指示（System Prompt）とエージェントの役割定義を1ファイルに統合して生成する。
- **ディレクトリ構造の構築**: `docs/dev` などの標準的なパスを作成する。
- **スキル導入のガイド**: `npx skills add` コマンドを使用したスキル導入のワークフローを提供する。
- **独立性と汎用性**: 特定の技術スタックや他スキルに依存せず、テンプレートとして機能する。

## 受入れ条件

- [ ] プロジェクトルートに `AGENTS.md` が生成されること。
- [ ] `AGENTS.md` 内にセクション「Project Instructions」と「Roles」が含まれていること。
- [ ] `docs/dev/` ディレクトリが作成されること。
- [ ] スキル追加の手順が提示されること。
