# QA Skill Test Cases

## Case 1: Issue Analysis
* **Description**: Analyze a bug report and suggest a fix.
* **Pre-conditions**: A codebase with a known bug.
* **User Input**:
  > このコードのバグを見つけて修正案を出してください。
  > [Buggy Code Snippet]
* **Expected Output**:
  * バグの原因特定
  * 修正コードの提示

## Case 2: Test Case Generation
* **Description**: Generate unit tests for a function.
* **Pre-conditions**: None.
* **User Input**:
  > 次の関数のユニットテストをJestで書いてください。
  > [Function Code]
* **Expected Output**:
  * Jest形式のテストコード
  * 正常系・異常系のカバレッジ
