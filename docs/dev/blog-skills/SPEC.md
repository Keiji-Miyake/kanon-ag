# Specification: Blog Generation Skills

## Overview
ユーザーが開発作業をアウトプット（ブログ記事等）に変換するプロセスを効率化するため、素材収集と執筆を専門化した2つのスキルを導入する。

## Features

### 1. tech-storyteller
- **目的**: リポジトリのコンテキスト（Git履歴、タスクログ、ドキュメント）を探索し、記事の素材を抽出する。
- **インプット**: プロジェクトディレクトリ、Git履歴。
- **アウトプット**: Markdown形式の素材集（事実、背景、解決策）。

### 2. blog-writer
- **目的**: 素材を高品質なブログ記事へと昇華させる。
- **機能**: 構成提案、リード文生成、SEO最適化、Mermaid図解。
- **インプット**: 任意の素材（メモ、tech-storytellerの出力等）。

## Design Principles
- **疎結合 (Decoupled)**: スキル同士を依存させず、標準的なMarkdownを通じて連携可能にする。
- **単一責任 (SRP)**: 収集と執筆の役割を厳格に分離する。
