# SPEC.md - スネークゲーム (Snake Game)

## Overview
ユーザーが楽しめるシンプルで古典的なスネークゲームをWebブラウザ上で作成します。レトロな雰囲気を持ちつつ、現代的なデザイン要素を取り入れ、ストレスなくプレイできることを目指します。

## User Stories
- ユーザーは、矢印キーまたはWASDキーを使用してヘビの進行方向を操作できる。
- ユーザーは、画面上のエサを食べることでヘビの体を長くし、スコアを上げることができる。
- ユーザーは、ヘビが自分自身の体や壁に衝突するとゲームオーバーになる。
- ユーザーは、ゲームオーバー後にリスタートボタンを押すことで、即座に新しいゲームを開始できる。

## Functional Requirements
- **グリッドベースの移動**: ヘビは一定の間隔（グリッド単位）で移動する。
- **エサの生成**: エサは画面内のランダムな位置（ヘビの体と重ならない場所）に生成される。
- **衝突判定**: 
    - 壁との衝突: 画面外に出た場合にゲームオーバー。
    - 自分自身との衝突: 頭が体の一部に重なった場合にゲームオーバー。
- **スコア管理**: エサを食べるごとに10ポイント追加される。
- **リスタート機能**: ゲームオーバー画面からワンクリックで初期状態に戻る。

## Non-Functional Requirements
- **レスポンシブデザイン**: デスクトップブラウザで快適に動作すること。
- **パフォーマンス**: 60FPS程度の滑らかな描画を維持すること。
- **アクセシビリティ**: キーボード操作のみでゲームの全機能が利用できること。
- **ビジュアル**: ダークモードを基調とした、プレミアムでネオンなデザイン。
